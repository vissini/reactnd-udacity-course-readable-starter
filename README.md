# Udacity nano-degree final project

Developed by Joan Peralta - [Github Repository URL](https://github.com/jgdev/reactnd-course-readable-starter)

Get started
===
* Install and start the API server
    - `cd api-server`
    - `npm install`
    - `node server`
* In another terminal window, run the following commands
    - `cd frontend`
    - `npm install`
    - `npm start`
    you also can run the program using the environment var `REACT_APP_BACKEND`.
